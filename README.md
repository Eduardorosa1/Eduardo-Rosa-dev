# Eduardo Rosa — QA & Back-end

Portfólio pessoal com foco em testes de software e desenvolvimento back-end.

Atuo validando comportamento de aplicações, principalmente APIs, autenticação e integração com banco de dados. O objetivo é simples: evitar erro em produção e garantir que o sistema funcione como deveria.


## Tecnologias

- JavaScript / Node.js
- React
- Cypress (testes automatizados E2E)
- APIs REST
- MySQL / PostgreSQL
- Git / GitHub


## Testes Automatizados

Utilizo Cypress para validar fluxo real de aplicação e evitar regressão.

Principais cenários que trabalho:
- Login (sucesso e falha)
- Interceptação de requisições (`cy.intercept`)
- Validação de resposta de API (status e dados)
- Fluxo completo de usuário (E2E)
